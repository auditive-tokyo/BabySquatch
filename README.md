# BoomBaby Project

## Overview

I built BoomBaby for one specific use case. I was a longtime fan of Sasquatch V1 (Boz Labs) — until I switched to an Apple M4 Pro Mac in mid-2025 and found it required Rosetta 2.

Kick Ninja (Him DSP) came up as an alternative — and honestly, it's a fantastic plugin. I especially love the pitch envelope; that's what gives it the freedom to shape really expressive sub movements. The only catch: it's a VST Instrument, not a VST Effect. For someone who frequently works with pre-bounced kicks in collabs and mixing sessions, routing through MIDI just to process an audio file was too much friction.

Rather than compromising, I analyzed both plugins by ear, extracted the features that actually mattered for my workflow, and built something streamlined around exactly that. The noise generation approach is inspired by Sasquatch's design — filtered noise for transient character, shaped to taste. If you need something more versatile, both are excellent commercial options. BoomBaby is intentionally focused — but it covers everything I need, and I hope it does the same for you.

I'm planning a commercial version with a broader feature set. This free version is released as open source, with the hope that it might be useful to talented beatmakers and producers around the world.
