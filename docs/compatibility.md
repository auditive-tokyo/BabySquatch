# Compatibility

## Platform Requirements

|                  |                                                         |
| ---------------- | ------------------------------------------------------- |
| **OS**           | macOS only (Windows / Linux not supported at this time) |
| **Architecture** | Apple Silicon (arm64) — Intel Macs are not supported    |
| **Formats**      | AUv2, VST3                                              |

> The distributed binary is arm64-only and is not a universal build.

---

Tested environments are listed below. If you have tested BoomBaby on a configuration not listed below, please share your results in [Compatibility Reports page](https://github.com/auditive-tokyo/BoomBaby/discussions/1) — your contribution will help the community. Thank you!
